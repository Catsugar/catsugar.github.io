# 不靠谱学习笔记---Typescript


>这是一个不靠谱的学习笔记。想到啥写啥



### 安装 compiler

``` bash
# install compiler
npm install -g typescript

```
装的时候被拒绝访问了，用管理员方式运行就好了。


### 编译

``` bash
# compile
cd path
tsc file

```
